## 🔄 Поток работы

### Регистрация клиента:
1. Клиент открывает Telegram бота
2. Отправляет `/start`
3. Вводит имя и телефон
4. Получает логин и пароль
5. Открывает веб-приложение через кнопку меню
6. Входит с полученными данными

### Создание заказа:
1. Клиент просматривает каталог
2. Добавляет товары в корзину
3. Оформляет заказ (заполняет адрес, телефон)
4. Заказ сохраняется в БД со статусом "new"
5. Админ получает уведомление в Telegram группу
6. Клиент получает уведомление в бот

### Обработка заказа:
1. Админ открывает админ-панель
2. Видит новый заказ
3. Меняет статус на "preparing"
4. Клиент получает уведомление "Ваш заказ готовится"
5. Админ меняет статус на "delivering"
6. Клиент получает уведомление "Ваш заказ доставляется"
7. Админ меняет статус на "delivered"
8. Клиент получает уведомление "Ваш заказ доставлен"

## 🚀 Деплой на Railway

1. Подключите GitHub репозиторий

5. Запустите миграции: `npm run migrate`
