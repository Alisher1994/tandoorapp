# Модуль «Конструктор Витрины» - Полная реализация

## 📋 Описание

Реализован полнофункциональный модуль конструктора витрины с динамической навигацией и управлением отображением категорий товаров для Telegram Mini App.

## 🎯 Основные функции

### 1. **Структура навигации (Client Side)**
- **Иконка «Витрина»** (🛍️) - новая главная страница (/)
- **Иконка «Меню»** (📋) - каталог с сеткой категорий (/catalog)
- **«Избранное»** (❤️) - без изменений
- **«Корзина»** (🛒) - без изменений
- **«Бронь»** (🪑) - если включено

### 2. **Интерфейс конструктора (для операторов)**
**Доступ:** `/admin/showcase`

#### Левая панель: Ресурсная база
- Список активных категорий с поиском
- Поиск/фильтр по названию категории
- Drag-and-drop категорий для добавления в блоки
- Отображение количества товаров в категории

#### Правая панель: Визуальный холст
- Кнопка «Добавить блок» с выбором типа
- Предпросмотр каждого блока в реальном времени
- Управление блоками (удаление, перемещение, настройка)
- Автосохранение с индикатором состояния

### 3. **Типы блоков**

#### Grid 3x (Маленькие иконки)
```json
{
  "block_type": "grid_3",
  "content": [id1, id2, id3, ...],
  "title": "Готовая еда"
}
```
- 3 колонны
- Идеально для быстрого просмотра
- Показывает картинку и название категории

#### Grid 2x (Средние карточки)
```json
{
  "block_type": "grid_2",
  "content": [id1, id2, ...],
  "title": "Категории"
}
```
- 2 колонны
- Вертальные карточки с описанием
- Традиционный вид Tandoor

#### Banner/Hero (На всю ширину)
```json
{
  "block_type": "banner",
  "title": "Спецпредложение",
  "settings": {
    "imageUrl": "https://...",
    "backgroundColor": "#667eea",
    "textColor": "#ffffff",
    "ctaText": "Подробнее"
  }
}
```
- Полная ширина экрана
- Для спецпредложений и акций
- Кастомизируемый фон и текст

#### Horizontal Slider (Товары)
```json
{
  "block_type": "slider",
  "category_id": 10,
  "title": "Популярное"
}
```
- Горизонтальная прокрутка товаров из одной категории
- Автоматическое заполнение товарами
- Идеально для "Новинки" или "Популярное"

### 4. **Логика работы**

#### Сортировка блоков
- Кнопки ↑/↓ для перемещения блоков
- Автоматическое обновление порядка (order: 0, 1, 2...)

#### Слайдер товаров
- В настройках блока выбирается категория
- Товары автоматически загружаются из этой категории
- При клике на категорию в слайдере → переход в меню с фильтром

#### Переходы между витриной и меню
- Клик на категорию в витрине → переход в /catalog с selectedCategoryId в state
- Каталог автоматически фильтруется по выбранной категории
- Бейджи с количеством товаров в корзине отображаются на карточках

### 5. **Визуальные требования**

#### UI/UX стиль
- ✅ Border-radius: 10px для блоков и карточек
- ✅ Бейджи с количеством товаров из корзины (красивые индикаторы)
- ✅ Адаптивность: 320px - 500px+
- ✅ Гладкие переходы и анимации
- ✅ Консистентность со стилем Tandoor

#### Responsive дизайн
- Автоматическое масштабирование под ширину экрана
- Grid-based layout для оптимального использования пространства
- Touch-friendly интерфейс для мобильных

## 🔌 API Endpoints

### GET /api/products/restaurant/:restaurantId/showcase
Получить макет витрины

**Ответ:**
```json
{
  "blocks": [
    {
      "id": "block_1234567890",
      "block_type": "grid_3",
      "title": "Готовая еда",
      "content": [1, 2, 3],
      "order": 0
    }
  ]
}
```

### POST /api/products/restaurant/:restaurantId/showcase
Сохранить макет витрины

**Тело запроса:**
```json
{
  "blocks": [
    {
      "id": "block_1234567890",
      "block_type": "grid_2",
      "title": "Категории",
      "content": [1, 2, 3, 4],
      "order": 0
    }
  ]
}
```

## 💾 Схема БД

### Таблица showcase_layouts
```sql
CREATE TABLE showcase_layouts (
  id SERIAL PRIMARY KEY,
  restaurant_id INTEGER NOT NULL UNIQUE,
  layout JSONB DEFAULT '[]'::jsonb,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
)
```

**Структура layout (JSON):**
```json
[
  {
    "id": "unique_block_id",
    "block_type": "grid_3|grid_2|banner|slider",
    "title": "Block Title",
    "content": [category_ids],
    "category_id": 10,
    "settings": { ... },
    "order": 0
  }
]
```

## 🚀 Использование

### Для пользователей
1. Откройте приложение → попадете на витрину (/)
2. Нажмите на категорию → откроется каталог с фильтром
3. Добавьте товары в корзину
4. Бейджи на карточках витрины обновляются в реальном времени

### Для операторов
1. Перейдите в админку → вкладка "Витрина"
2. Или прямой доступ: `/admin/showcase`
3. Слева: выберите категории из ресурсной базы
4. Справа: добавляйте блоки, перетаскивайте категории
5. Используйте кнопки для управления и перемещения
6. Нажмите "Сохранить" → витрина обновляется для всех пользователей
