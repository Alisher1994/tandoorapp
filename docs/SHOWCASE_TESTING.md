# Модуль Витрина - Примеры и Тестирование

## 🎬 Примеры использования

### Пример 1: Создание витрины с Grid 3x блоком

```javascript
// ShowcaseContext методы:
const { addBlock, addCategoryToBlock, saveShowcase } = useShowcase();

// Добавить блок Grid 3x
const block = addBlock('grid_3', {
  title: 'Готовая еда',
  content: []
});

// Добавить категории в блок
addCategoryToBlock(block.id, 1); // Пицца
addCategoryToBlock(block.id, 2); // Бургеры
addCategoryToBlock(block.id, 3); // Салаты

// Сохранить витрину
await saveShowcase(restaurantId, showcaseLayout);
```

### Пример 2: Создание витрины с Slider блоком

```javascript
const { addBlock, setSliderCategory } = useShowcase();

// Добавить слайдер
const sliderBlock = addBlock('slider', {
  title: 'Популярное',
  category_id: null
});

// Выбрать категорию для слайдера
setSliderCategory(sliderBlock.id, 5); // Категория "Напитки"

await saveShowcase(restaurantId, showcaseLayout);
```

### Пример 3: Полная витрина с несколькими блоками

```javascript
const { addBlock, addCategoryToBlock, setSliderCategory, reorderBlocks, saveShowcase } = useShowcase();

// 1. Grid 3x блок
const grid3 = addBlock('grid_3', { title: 'Быстрое' });
addCategoryToBlock(grid3.id, 1);
addCategoryToBlock(grid3.id, 2);

// 2. Banner блок
const banner = addBlock('banner', {
  title: 'Скидка 30%',
  settings: {
    imageUrl: 'https://example.com/promo.jpg',
    backgroundColor: 'linear-gradient(135deg, #FF6B6B 0%, #FFD93D 100%)',
    textColor: '#ffffff',
    ctaText: 'Заказать'
  }
});

// 3. Slider блок
const slider = addBlock('slider', { title: 'Популярное' });
setSliderCategory(slider.id, 3);

// 4. Grid 2x блок
const grid2 = addBlock('grid_2', { title: 'Все категории' });
addCategoryToBlock(grid2.id, 1);
addCategoryToBlock(grid2.id, 2);
addCategoryToBlock(grid2.id, 3);
addCategoryToBlock(grid2.id, 4);

// Сохранить все блоки
await saveShowcase(restaurantId, showcaseLayout);
```

### Пример 4: Изменение витрины

```javascript
const { 
  showcaseLayout, 
  removeBlock, 
  reorderBlocks, 
  updateBlock, 
  saveShowcase 
} = useShowcase();

// Найти первый блок
const firstBlock = showcaseLayout[0];

// Переместить блок вниз
reorderBlocks(firstBlock.id, 'down');

// Обновить название блока
updateBlock(firstBlock.id, { title: 'Новое название' });

// Удалить последний блок
if (showcaseLayout.length > 0) {
  removeBlock(showcaseLayout[showcaseLayout.length - 1].id);
}

// Сохранить
await saveShowcase(restaurantId, showcaseLayout);
```

## 🧪 Тестирование

### Шаг 1: Проверка базовой функциональности

#### Тест 1.1: Витрина отображается как главная страница
```
1. Откройте приложение
2. Должны увидеть витрину (/) с блоками
3. В bottomNav должна быть иконка "Витрина" (🛍️)
```

#### Тест 1.2: Загрузка конструктора
```
1. Войдите как оператор
2. Перейдите в /admin
3. Найдите вкладку "Витрина"
4. Нажмите ссылку "Открыть конструктор"
5. Откроется страница /admin/showcase
```

### Шаг 2: Тестирование конструктора

#### Тест 2.1: Добавление блока Grid 3x
```
1. На странице /admin/showcase нажмите "Добавить блок"
2. Выберите "Сетка 3x (3 колонны)"
3. Появится новый блок Grid 3x в canvas
4. Слева видна ресурсная база с категориями
```

#### Тест 2.2: Drag-and-drop категорий
```
1. В левой панели найдите категорию
2. Перетащите ее на блок Grid 3x
3. Категория должна добавиться в блок
4. На превью блока должна появиться категория
```

#### Тест 2.3: Сортировка блоков
```
1. Добавьте несколько блоков
2. Для первого блока нажмите кнопку ↓
3. Блоки должны поменяться местами
4. Для второго (теперь первого) блока нажмите ↑
5. Блоки должны вернуться в исходный порядок
```

#### Тест 2.4: Удаление блока
```
1. Нажмите иконку 🗑️ на любом блоке
2. Блок должен удалиться
3. Обновится счетчик блоков
```

#### Тест 2.5: Сохранение витрины
```
1. Создайте несколько блоков
2. Нажмите "Сохранить"
3. Должно появиться сообщение об успехе
4. Обновите страницу - блоки должны остаться
```

### Шаг 6: Тестирование производительности

#### Тест 6.1: Время загрузки
```
1. Откройте DevTools → Network
2. Загрузите витрину (/)
3. Время загрузки должно быть < 2 сек
4. API запрос к /api/products/restaurant/:id/showcase должен < 500ms
```

#### Тест 6.2: Плавность прокрутки
```
1. На витрине прокручивайте вверх-вниз
2. Прокрутка должна быть плавной (60 fps)
3. Нет фризов и задержек
```

#### Тест 6.3: Быстрое переключение между навигацией
```
1. Быстро нажимайте на разные вкладки в bottomNav
2. Переходы должны быть мгновенными
3. Нет задержек при загрузке страниц
```

### Шаг 7: Edge cases

#### Тест 7.1: Пустая витрина
```
1. Удалите все блоки из витрины
2. Сохраните
3. Откройте витрину пользователя
4. Должно видеть сообщение "Витрина не настроена"
```

#### Тест 7.2: Много блоков
```
1. Добавьте 15-20 блоков
2. Сохраните
3. Откройте витрину
4. Прокрутка должна быть плавной
5. Все блоки должны загрузиться
```

#### Тест 7.3: Очень длинные названия
```
1. Создайте блок с очень длинным названием
2. Добавьте категорию с длинным названием
3. Текст должен быть обрезан (ellipsis)
4. Не должно быть overflow текста
```

#### Тест 7.4: Отсутствие изображений категорий
```
1. Если категория без изображения
2. Должно видеть серый placeholder
3. Названия должны быть видны
4. Блоки должны правильно выравниваться
```

## 🐛 Troubleshooting

### Проблема: Витрина не загружается
**Решение:**
1. Проверьте браузерную консоль на ошибки
2. Проверьте Network tab - есть ли 404/500 ошибки
3. Убедитесь, что ShowcaseProvider обернут в App.jsx
4. Перезагрузите приложение (Ctrl+Shift+R)

### Проблема: Блоки не сохраняются
**Решение:**
1. Проверьте наличие showcase_layouts таблицы в БД
2. Проверьте API endpoint в browser console
3. Убедитесь, что вы авторизованы как оператор
4. Посмотрите ошибку на сервере (server logs)

### Проблема: Категории не добавляются в блок
**Решение:**
1. Убедитесь, что вы перетаскиваете на блок, а не вне его
2. Проверьте, что категория активна (у нее есть товары)
3. Обновите страницу конструктора

### Проблема: Бейджи не обновляются
**Решение:**
1. Убедитесь, что CartContext работает правильно
2. Добавьте товар и проверьте консоль
3. Перезагрузите витрину (нажмите F5)
