## 🔄 Структура данных - Block Object

```javascript
// Block структура (JSON)
{
  id: "block_1234567890",        ← уникальный ID
  block_type: "grid_3",           ← тип: grid_3, grid_2, banner, slider
  title: "Готовая еда",           ← название для показа
  content: [1, 2, 3],             ← ID категорий (для grid и slider)
  category_id: null,              ← для slider блока
  settings: {                     ← специфичные для типа настройки
    imageUrl: "https://...",
    backgroundColor: "#667eea",
    textColor: "#ffffff",
    ctaText: "Подробнее"
  },
  order: 0                        ← порядок отображения (0, 1, 2...)
}
```

## 🖱️ Взаимодействие в конструкторе

```
1. DRAG-AND-DROP КАТЕГОРИЙ
   ┌─────────────────────┐     ┌──────────────────┐
   │ Левая панель        │     │ Правая панель    │
   │ (Ресурсная база)    │     │ (Холст)          │
   │                     │     │                  │
   │ [Категория 1] ─────────→ [Блок Grid 3x]    │
   │       ↓ (drag)      │     │                  │
   │       │ (drop)      │     │ [Категория 1]    │
   │       └─────────────────→ [Категория 2]    │
   └─────────────────────┘     └──────────────────┘

2. СОРТИРОВКА БЛОКОВ
   Блок 1 [↑][↓]  → блоки меняются местами
   Блок 2 [↑][↓]

3. КНОПКИ УПРАВЛЕНИЯ
   [⚙️] Открывает настройки блока (название, цвета, категория)
   [🗑️] Удаляет блок

4. КНОПКА СОХРАНЕНИЯ
   [Сохранить] → отправляет на сервер → витрина обновляется для всех
```

## 🎯 User Journey - Сценарий использования

### Сценарий 1: Оператор создает витрину

```
1. Оператор входит в админку
2. Открывает вкладку "Витрина"
3. Переходит в конструктор (/admin/showcase)
4. Добавляет блок Grid 3x
5. Перетаскивает категории в блок
6. Добавляет Banner блок со спецпредложением
7. Добавляет Slider с популярными товарами
8. Нажимает "Сохранить"
9. ✅ Витрина сохранена в БД
```

### Сценарий 2: Пользователь просматривает витрину

```
1. Пользователь открывает приложение
2. Видит витрину (/) с 4 блоками
3. Прокручивает витрину вверх-вниз
4. Видит категории с картинками
5. Видит спецпредложение
6. Видит слайдер с товарами
7. Нажимает на категорию
8. Переходит в каталог (/catalog) с фильтром
9. Добавляет товар в корзину
10. Видит бейдж на витрине
11. Продолжает делать покупки 🛒
```

### Сценарий 3: Пользователь скользит по слайдеру

```
1. Пользователь видит блок "Популярное"
2. Видит товары в горизонтальном слайдере
3. Скользит влево → видит новые товары
4. Нажимает на товар → добавляет в корзину
5. Видит число на бейдже слайдера
```
