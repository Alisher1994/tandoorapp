# Talablar Agent Agent (Windows)

Локальный агент для автоматической печати чеков на термопринтерах XP-80T (и совместимых).

## Установка

1. Установите [Node.js](https://nodejs.org/) (рекомендуется LTS версия).
2. Скачайте эту папку `printer-agent` на компьютер, к которому подключен принтер.
3. Откройте терминал (PowerShell или CMD) в этой папке.
4. Выполните команду:
   ```bash
   npm install
   ```
5. Создайте файл `.env` в этой папке и укажите настройки:
   ```env
   SERVER_URL=https://ВАШ_ДОМЕН_ИЛИ_IP:3000
   AGENT_TOKEN=ВАШ_ТОКЕН_ИЗ_АДМИНКИ
   ```

## Запуск

```bash
npm start
```

## Смена токена / смена магазина

- Если агент уже установлен и токен нужно поменять, запустите:
  ```bash
  node agent.js --setup
  ```
- В установленной версии через `.exe`:
  ```bat
  TalablarAgent.exe --setup
  ```
- Это откроет мастер, где можно сменить `SERVER_URL` и `AGENT_TOKEN` без ручного редактирования `.env`.

## Запуск в трее Windows

- Используйте `start-tray.bat` (или установленный ярлык `Запустить Принтер (Трей)`).
- Агент работает в фоне через иконку в системном трее, поэтому окно `cmd` не закроют случайно.

## Как это работает

1. Агент подключается к серверу Talablar через WebSocket.
2. При подтверждении заказа кассиром (в админке), сервер отправляет команду печати.
3. Агент автоматически распределяет товары по принтерам (Кухня, Бар, Касса) согласно настройкам в админке.
4. Печатается профессиональный чек со всеми данными клиента.

## Настройка принтеров

В админке Talablar (раздел "Принтеры"):
- **Master/Cashier**: Основной принтер у кассира (печатает полный чек).
- **Kitchen/Bar**: Принтеры для цехов (печатают только нужные товары).

Для USB принтеров укажите `VID,PID` (например, `0x04b8,0x0202`). 
Для сетевых — IP адрес (например, `192.168.1.100`).

## Важно
- Для USB принтеров на Windows может потребоваться установка драйвера **Zadig** (выберите принтер и замените текущий драйвер на `WinUSB`).
- Убедитесь, что порт 9100 открыт, если используете сетевой принтер.
