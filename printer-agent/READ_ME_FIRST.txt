================================================================
          ИНСТРУКЦИЯ ПО УСТАНОВКЕ ПРИНТЕРА (ДЛЯ ОПЕРАТОРА)
================================================================

ВАЖНО: оператору больше не нужен Node.js и ручные команды.

КАК ЗАПУСТИТЬ:

1. Откройте папку [dist].
2. Запустите [install.bat].
3. Введите:
   - Server URL
   - Agent Token (из админки -> Принтеры -> Токены агентов)
4. После установки запускайте через ярлык [Talablar Printer].
   Агент работает в системном трее.

КАК СМЕНИТЬ МАГАЗИН / ТОКЕН (НА ЭТОМ ЖЕ ПК):

1. Правый клик по иконке Talablar Agent в трее
2. "Настройки (сервер/токен)"
3. Вставьте новый токен и сохраните
4. Агент перезапустится автоматически

ЕСЛИ НЕ ПЕЧАТАЕТ:

1. Проверьте, что принтер подключен и включен
2. Для USB-принтера включите "Общий доступ" в Windows
3. Убедитесь, что в админке выбран правильный принтер для точки
