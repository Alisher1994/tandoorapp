================================================================
          ИНСТРУКЦИЯ ПО УСТАНОВКЕ ПРИНТЕРА (ДЛЯ ОПЕРАТОРА)
================================================================

Я подготовил "Быстрый Установщик", чтобы вам не нужно было 
пользоваться командной строкой вручную. 

ШАГИ ДЛЯ ЗАПУСКА:

1. Убедитесь, что у вас установлен Node.js (https://nodejs.org/).
   (Если нет - скачайте и установите версию LTS).

2. Запустите файл [ setup.bat ] в папке printer-agent.
   Он сам всё установит.
   - Он спросит "Server URL" (адрес вашего сайта).
   - Он спросит "Agent Token" (токен, который вы создали в админке).

3. Если у вас USB принтер (например XP-80):
   - Зайдите в "Устройства и принтеры" в Windows.
   - Щелкните правой кнопкой на ваш принтер -> Свойства принтера.
   - Перейдите во вкладку "Доступ" (Sharing).
   - Поставьте галочку "Общий доступ к данному принтеру".
   - Укажите сетевое имя (например: XP-80).
   - В админке сайта для этого принтера укажите тип соединения "usb" 
     и в поле для VID/PID просто напишите это имя: XP-80.

4. Для запуска принтера используйте файл [ start.bat ].

ПРЕИМУЩЕСТВА:
- Больше не будет ошибок при npm install (я убрал проблемные модули).
- Не нужны специальные драйверы Zadig (достаточно общего доступа Windows).
- Всё настраивается через простые вопросы в установщике.
